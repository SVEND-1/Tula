## 🛠 Стек технологий

### Backend
- Java 21
- Spring Boot 3.2.4
- Spring Security + JWT
- Spring Data JPA / Hibernate
- PostgreSQL
- Apache Kafka
- MinIO (S3-совместимое хранилище)
- Maven
- Docker / Docker Compose
- ЮКАССА

### Frontend
- React
- TypeScript
- Axios
- React Router
- CSS / HTML / SCSS

### DevOps & Деплой
- Nginx (прокси)
- Docker / Docker Compose
